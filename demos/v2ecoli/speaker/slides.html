Slide 11 · Recap · 20:00–21:00

Questions?

Anticipated:

  • Do I need to be a v2ecoli expert? No — dashboard is simulator-agnostic.
  • How do I add my own simulator? pip install → workspace.yaml → refresh.
  • What if my sim takes hours? Remote pipeline offloads to AWS.
  • Is it open source? MIT — vivarium-collective/vivarium-workbench.
  • How do I share results? Push branch, export HTML, or static bundle.
